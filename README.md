# G&D Light Codex - Star Wars Datapad Interface

A Star Wars datapad-styled website for the G&D Light rules system.

## Features

- **Star Wars Datapad Aesthetic**: Dark theme with glowing cyan borders and futuristic UI
- **Interactive Navigation**: Tab-based navigation between sections
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Two Main Sections**:
  - **Création de personnage**: Character creation guide with 4 steps
  - **Gloire au MJ**: Game Master resources and tips

## How to Use

1. Open `index.html` in a web browser
2. Navigate between sections using the top navigation buttons
3. You can also use arrow keys for keyboard navigation

## Technologies

- Pure HTML5, CSS3, and JavaScript
- No frameworks or dependencies required
- Works offline

## Browser Compatibility

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers
